## Goal
Make the repo a loadable Claude Code plugin and adopt the final namespaced command set in one task (they're coupled — the invocation name is plugin-name + skill/command-name).

- Add `.claude-plugin/plugin.json` — `name: flow`, `version`, `description`, `author`, `homepage`/`repository`. Omit nothing the marketplace needs.
- Add `.claude-plugin/marketplace.json` listing the `flow` plugin (this repo as its own marketplace source).
- Rename (git mv, preserve history):
  - skills: `skills/flow`→`skills/run`, `skills/flow-hunt`→`skills/hunt`, `skills/flow-review`→`skills/review`, `skills/flow-pr`→`skills/pr`; set each `SKILL.md` `name:` to the new bare name.
  - commands: `commands/flow-init.md`→`commands/init.md`, `flow-lint.md`→`lint.md`, `flow-ship.md`→`ship.md`.
- Update every internal cross-reference to the renamed entry points (skill bodies referencing `/flow`, `/flow-hunt`, `/flow-lint`, `/flow-ship`, `reference/*` mentions, `flow-preflight.sh` locate-instructions that name commands, agent prose). Agents keep `flow-*` file/`name:` — but confirm the skills still dispatch them.
